## JavaFX+Mysql实现的一个图书后台管理系统
**1.技术栈**
- Java
- JavaFX
- Mysql
- maven

**2.功能**
- 登录
- 首页dashbord：同步系统时间、图片轮播、数据统计、图表展示
- 图书类别管理：增、删、查
- 图书信息管理：增、删、查（所有、按类别、模糊查）、图书浏览、数据导出
- 图书数据统计分析：饼图、柱状图
- 读者信息管理：增、删、查
- 读者数统计分析：饼图、柱状图
- 管理员信息：查
