## JavaFX+Mysql实现的一个图书后台管理系统
**1.技术栈**
- Java
- JavaFX
- Mysql
- maven

**2.功能**
- 登录
- 图书类别管理：增、删、查
- 图书信息管理：增、删、查（所有、按类别、模糊查）、图书浏览
- 图书数据统计分析：柱状图、饼图
- 读者信息管理：增、删、查
